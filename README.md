# Rubric

A lightweight CLI tool for fetching and displaying HTTP headers from any URL, written in Go.

## Features

- Fetch HTTP headers from any URL
- Display HTTP status and response headers
- Simple, clean output format

## Usage

```bash
go run main.go
```

## Building

```bash
go build
./rubric
```

## Requirements

- Go 1.25.4 or later

## Module

`github.com/niranx/rubric`
